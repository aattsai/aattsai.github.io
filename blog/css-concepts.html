CSS Concepts - Border, Margin, and Padding

Time flies, this is already the end of week three for our phase 0 portion at Dev Bootcamp. It also means it has been three weeks since I moved to New York City. After two weeks of attempting to master the basic concepts behind html and css, I would like to write up a little bit of what I've learned about basic CSS concepts.

One of the more confusing concepts for beginners of CSS is the difference between Border, Margin, and Padding when one tries to implement an element onto a webpage. When I tried to understand the differences between these three properties, I found it extremely helpful to look at the image shown above as it provides a clear visualization of what each of the properties defines on a element, such as an image, a block of texts, or even a link, on your webpage. I find border to be the clearest to define since it is the line that essentially separates padding and margin. Border separates the inner and outer part of an element, and can be dotted, dashed, or solid.

Margin is defined as the area around the element, outside of the border. Since margin is not part of the element, it does not have a background color, and is transparent. So the larger a margin is, the farther apart the element is from neighboring elements. Margin can be adjusted using properties margin, which changes all margin on four sides all at once. One can also change the margin on a specific side using margin-left, margin-right, margin-top, or margin-bottom.

On the other hand, the space between the border and the content of the element, such as texts, is represented by padding. Since padding is an area inside the border, it does get affected by the background color assigned to that element. The larger the padding is, the smaller the content will appear within the element. Similar to margin, padding can be adjusted using properties padding, which changes all padding on four sides all at once. One can also adjust the padding on a specific side using padding-left, padding-right, padding-top, or padding-bottom.
