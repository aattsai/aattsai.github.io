II. JavaScript Objects

Literal Notation:

var object_example = {
   property_key1: "value1",
   property_key2: "value2",
   property_key3: "value3"
};

Constructor Notation

var object = new Object();
object.property_key1 = "value1";
object.property_key2 = "value2";
object.property_key3 = "value3";

Update an Object

object.proper_key1 = "new_value1";
object['proper_key1'] = "new_value1";

*Bracket notation is more expressive than dot notation because it allows a variable to specify all or part of the property name. This is possible because the JavaScript interpreter automatically converts the expression within the square brackets to a string, and then retrieves the corresponding property.

Create Multiple Instances of an Object

function Car(brand, color) {
   this.brand_name = brand;
   this.car_color = color;
};
var myCar = new Car("Saab", "black");
var herCar = new Car("Mercedes", "silver");


III. Array & Object Iteration

Iteration Array & Obejct Using For Loops

for (var i = 0 ; i < array.length ; i++) {
   array[i] += 2;
};

for (var property_key in object) {
   object.property_key = "new value";
};

Access Nested Objects

for (var property_key in Outer_Object) {
   for (var property_key2 in Outer_Object[property_key]) {
      Outer_Object[property_key][property_key2] += 1;
   }
}
